软件的生命周期?
+ 问题定义、可行性分析、总体描述、系统设计、编码、调试与测试、验收与运行、维护
软件测试是什么？
+ 使用技术手段验证软件是否满足需求
测试的类型
+ 功能测试、自动化测试、接口测试、性能测试
测试分类
+ 按照阶段划分，分为单元测试、集成测试、系统测试、验收测试
+ 按照代码可见性划分，黑盒测试、灰盒测试、白盒测试
质量模型重点五项
+ 安全、兼容性、功能性、性能、易用性
测试流程
+ 需求评审、测试计划、用例设计、用例执行、缺陷管理、测试报告
测试用例八大标题
+ 测试编号、测试标题、项目/模块、优先级、前置条件、测试步骤、测试数据、预期结果
bug/缺陷的生命周期
+ 需求规格说明、软件设计、编码、测试、故障分类、故障隔离、故障解决
